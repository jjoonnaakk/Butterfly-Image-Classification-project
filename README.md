# Butterfly Image Classification

A deep learning project that classifies butterfly species using Convolutional Neural Networks (CNN).

## Features

- Image preprocessing
- CNN model
- Training and validation
- Prediction on new images

## Technologies

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas

## Dataset

Dataset from Kaggle.

## Results

Training Accuracy: 70%

## Installation

pip install -r requirements.txt



python train.py

python predict.py
